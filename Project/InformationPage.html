Информация о сайте

Данный сайт представляет собой платформу для развития своих навыков программирования. Также можно померится силами с другими программистами. На сайте предусмотрены: выбор языка программирования, удобный интерфейс и возможность для каждого стать модератором сайта!При одиночном решении пользователь публикует свой код на оценку пользователей и экспертов. При выборе дуэльной игры все те же функции, только на обсуждение идут 2 кода, победитель озвучивается через неделю, ему насчитываются дополнительный рейтинг в зависимости от сложности задачи. Решение задач осуществляется в специальном поле, рядом есть обязательный выбор языка («Java/ Python/ Pascal/ Puby/ C/ C#/ C++»). Потом код отправляется на компиляцию. После компиляции полученные данные возвращаются на сайт и публикуются, далее оцениваются пользователями. При дуэли на оценку выходят два кода одновременно, после решения обоих оппонентов. Баллы пользователей при оценке решенной задачи идут в рейтинг пользователя. При достижении определенного рейтинга пользователь становится экспертом, и его оценка становится выше. Количество экспертов мало, им дается возможность выложить свои задачи. Стать экспертом можно, победив эксперта в длительной дуэли 2-4 задачи. Если эксперт долгое время не активен, он лишается своего «Титула»
